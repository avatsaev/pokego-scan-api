# PokeGo Scan API

Lighweight Scan API for Pokemon Go

### Version
1.0.0


### Configuration

Edit **config.json** with your account credentials

### Installation

PokeGo Scan API requires [Node.js](https://nodejs.org/) v4+ to run.


Install the dependencies and start the server.

```sh
$ npm install
$ npm start
```


### Endpoints

**GET: /scan.json**

Params:

    - lat: latitude
    - lon: longitude
